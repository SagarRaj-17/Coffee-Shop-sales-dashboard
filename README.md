# ☕ Coffee Shop Sales Dashboard

# Overview

This project is an interactive Power BI dashboard developed to analyze coffee shop sales data using SQL, DAX, Excel and Power BI.
The dashboard helps business users monitor sales performance, identify trends, compare stores performance, and analyze customer purchasing behavior.

# Tools Used

- Power BI
- SQL (MySQL)
- DAX
- Microsoft Excel

# Dashboard Features

- Total Sales KPI
- Total Orders KPI
- Total Quantity Sold
- Daily Sales Trend
- Sales by Product Category
- Sales by Product Type
- Sales by Store Location
- Weekday vs Weekend Analysis
- Sales by Hour Heatmap
- Month-over-Month Growth

# Key Insights

- Coffee generated the highest revenue.
- Morning hours recorded the highest sales.
- Weekends outperformed weekdays.
- Store performance varied across locations.
